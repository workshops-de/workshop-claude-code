## Learning goals

- **Skills:** delegating research/verification to a subagent; constraining a subagent's tools;
  reading its summary.
- **Concepts:** delegation; context isolation; orchestration vs. execution.
- **Takeaways:** push noisy work out of the main context; ask for a concise report back.

## Facilitation notes

- The before/after context comparison (direct ask vs. delegated) is the moment that sells this
  topic — make sure everyone actually runs both, not just one.
- If time allows, briefly demo agent teams / worktrees conceptually even if the room doesn't build
  it hands-on.

## Time estimate

~30 minutes.

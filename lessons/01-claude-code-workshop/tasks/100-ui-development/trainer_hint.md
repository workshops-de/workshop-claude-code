## Learning goals

- **Skills:** building a vertical feature slice; reusing a read/write pattern; wiring validated
  forms.
- **Concepts:** server-first reads; server actions for writes; shared validation; path
  revalidation.
- **Takeaways:** define the feature shape once; reuse it everywhere; verify each screen.

## Facilitation notes

- This is the biggest single task so far — consider live-building the first entity together as a
  group before participants do the second entity solo.
- Watch for participants who skip "verify after each write" and only notice a stale list at the
  very end.

## Time estimate

~60 minutes; the "second entity" step can be trimmed if the group is running behind.

## Going further — governance & scale

- **Managed MCP** — restrict which servers users may add or connect to with allowlists and
  denylists.
- **Tool search** — scale to thousands of tools by discovering and loading only what's needed, on
  demand.
- **Channels** — push external events (CI results, alerts, chat messages) into a running session
  via an MCP server.
- **MCP in the Agent SDK** — wire servers into programmatic agents.

Try imagining an allowlist policy for MCP servers in your own organization.

## Learning goals

- **Skills:** running longer tasks with fewer prompts; setting a goal/condition; isolating runs in
  a worktree or sandbox; designing fail-safe autonomy.
- **Concepts:** semi-autonomous workflows; goal-directed execution; background and long-running
  tasks; unattended execution and its risks.
- **Takeaways:** autonomy is bounded delegation; isolate it, bound it, and review the outcome.

## Facilitation notes

- This opens the final morning — frame it as "everything you've learned, applied with less
  hand-holding."
- If auto mode isn't available on participants' accounts, a curated `--allowedTools` set works
  fine as a substitute — don't let access differences block the exercise.

## Time estimate

~30 minutes.

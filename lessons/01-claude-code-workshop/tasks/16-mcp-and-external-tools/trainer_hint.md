## Learning goals

- **Skills:** configuring an MCP server; having the agent use external tools; judging context
  cost.
- **Concepts:** tool discovery via MCP; closing the loop on external systems; context budgeting.
- **Takeaways:** extend reach with MCP, but prefer the leaner tool when context is tight.

## Facilitation notes

- This closes "Claude Code Building Blocks" — a good moment to recap the 2x4 matrix (Memory /
  Instructions / Execution / Delegation) covering skills, commands, subagents, hooks, and MCP
  together.
- Network/firewall issues can block MCP server installs in some environments — have a fallback
  screenshot/recording ready.

~30 minutes.

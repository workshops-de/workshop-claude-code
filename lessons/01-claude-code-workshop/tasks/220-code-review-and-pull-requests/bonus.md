## Going further — deeper, automated review

- **`/code-review`** and **ultrareview** run deep, multi-agent reviews (ultrareview runs in the
  cloud) to find and verify bugs before you merge.
- **GitHub Actions / GitLab CI** run Claude reviews automatically on every PR.
- **security-guidance plugin** — have Claude review its own changes for vulnerabilities and fix
  them in the same session.

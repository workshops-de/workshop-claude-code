## Learning goals

- **Skills:** bootstrapping and curating project memory; scoping memory to subdirectories.
- **Concepts:** long-term vs. short-term memory; signal-over-volume; memory as context cost.
- **Takeaways:** memory is curated; keep it lean; verify it actually changes agent behaviour.

## Facilitation notes

- Participants often keep too much of the `/init` output. Push them to cut aggressively — the
  "new teammate" test in the hints is a good prompt to repeat out loud.
- The fresh-session verification step (asking a question that depends on the memory file) is the
  step people skip under time pressure — don't let them; it's the only proof the file works.
- Good moment to reconnect to Task 1's "context is king" idea: this is the first concrete lever
  they get to pull.

## Time estimate

~30 minutes for an experienced group.

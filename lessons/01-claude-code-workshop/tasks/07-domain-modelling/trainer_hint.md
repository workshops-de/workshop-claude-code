## Learning goals

- **Skills:** using the agent to clarify requirements; building a shared glossary; modelling
  entities, relationships, and states.
- **Concepts:** ubiquitous language; entities vs. value objects; lifecycle/state of an entity.
- **Takeaways:** align on meaning before code; record the glossary where the agent can see it.

## Facilitation notes

- Participants eager to start coding may want to skip this — hold the line, this is the task that
  prevents the most expensive rework in the whole app-building arc.
- A good group discussion: compare glossaries across pairs/individuals and see where wording
  diverges even on the same brief.

## Time estimate

~30 minutes.

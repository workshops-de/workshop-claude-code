<details>
<summary>💡 Hint 1: Start from what you already type</summary>

A prompt you already type often is your first command candidate.

</details>

<details>
<summary>💡 Hint 2: Arguments add flexibility</summary>

Use arguments where the action varies (e.g. a scope or focus area).

</details>

<details>
<summary>💡 Hint 3: Who decides, you or the agent?</summary>

If the agent should decide _when_ to act, that's a skill; if you decide, that's a command.

</details>

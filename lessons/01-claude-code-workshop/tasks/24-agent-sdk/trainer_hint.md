## Learning goals

- **Skills:** installing and running the Agent SDK; constraining tools/permissions; returning
  structured output; loading `.claude/` features programmatically.
- **Concepts:** embedding Claude Code as a library; reusable agents; SDK vs. CLI vs. Client SDK vs.
  Managed Agents.
- **Takeaways:** the CLI and SDK share one engine; choose the SDK for custom apps and production
  automation.

## Facilitation notes

- Have both a Python and a TypeScript quickstart ready since the room will likely be mixed.
- This is a good moment to clarify the SDK/CLI/Client-SDK/Managed-Agents landscape at a high level
  if participants ask about production deployment options.

## Time estimate

~30 minutes.

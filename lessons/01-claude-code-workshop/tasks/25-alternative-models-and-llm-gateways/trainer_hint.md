## Learning goals

- **Skills:** identifying provider/gateway options; reading the configuration env vars; reasoning
  about governance trade-offs.
- **Concepts:** API compatibility surface; gateways and proxies; model routing and discovery;
  enterprise providers.
- **Takeaways:** "alternative models" means Claude via another provider/proxy; gateways add
  governance at the cost of an extra moving part.

## Facilitation notes

- Short, discussion-heavy task by design — good pace-recovery point before the two heavier
  methodology tasks that close the workshop.
- If anyone in the room works at an enterprise with existing Bedrock/Vertex/Foundry usage, invite
  them to share real constraints for the group discussion.

~25 minutes.

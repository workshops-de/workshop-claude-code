## Going further — composable, scoped & shared skills

- **Per-directory skills** load only when working in that area; the `paths` frontmatter scopes a
  skill by file glob.
- **Package skills as plugins** for versioned, team-wide, cross-repo reuse with namespacing.
- **Keep descriptions discoverable** — names always load, but descriptions are shortened when
  there are many, so lead with trigger words.

Try scoping your skill to a specific directory with the `paths` frontmatter field.
